# TRANSFORMACIONES - MA-0001 Precálculo

Material didáctico interactivo para la Universidad de Costa Rica - Escuela de Matemática.

**Tema:** Graficación con Transformaciones de funciones polinomiales

## Contenido

- **index.html** - Sitio completo (tipos, explorador interactivo, ejemplos, práctica)
- **explorer.html** - Explorador interactivo simplificado 
- **plotly.min.js** - Librería Plotly para gráficas interactivas (incluida localmente)

## Características

✅ Gráficas interactivas (zoom, pan, descarga)  
✅ Controles en tiempo real (sliders para transformaciones)  
✅ Análisis de transformaciones paso a paso  
✅ Ejemplos resueltos  
✅ Ejercicios de práctica

## Acceso

- **index.html** - Página principal completa
- **explorer.html** - Explorador de transformaciones

## Tecnologías

- HTML5 + CSS3
- JavaScript vanilla
- Plotly.js para visualización
- MathJax para ecuaciones matemáticas
